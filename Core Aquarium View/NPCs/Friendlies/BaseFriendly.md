# BaseFriendly

This serves as a intial parent class for all friendly fish
The purpose of this is to create friendlies from this template.

The BaseFriendly shall include the following features:
- HP (int)
- Mana (int)
- Origin (String)
- Movement Stats (Speed, Acceleration, Drag, X/Y Enable Checkbox)
- Ability Timer
- Hunger Timer
- Coin Timer
