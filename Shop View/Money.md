# Money

Currency Name: Gold Coins

## In-game Description

The fantasy world's universal currency! Gold is king!
Coins are manufactured by Dragons. Protected with mana inscriptions to prevent forgery.

## Where to get gold coins?

- Complete requests from customers
- Generate coins based on visitors + aquariums in the store

## Uses

- Aquarium Decor
- Store Decor
- New Store Layouts
- Hire Aquarium Helpers (Like the shrimp)
- Unlock New Content
