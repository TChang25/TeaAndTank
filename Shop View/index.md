# Shop View

The player's homebase. Personally designed tanks are on display, and customers walk around browsing the tanks.
Shop Layouts can be bought and a player can have multiple shops at the same time.

A shop can produce jewels while the player is away managing tanks

## Visitors

Visitors include all races the player has unlocked. As the player gains fame for his aquarium handling skills,
more races come into the store. 

## Accepting/Declining Requests

A request pops up from an visitor occassionally. The visitor will have a question mark above their head, indicating a request is awaiting the player.
There can be multiple visitors awaiting to be serviced.
