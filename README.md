# TeaAndTank

The objective of this repository is to document and share vision between team members.
Team members can begin by reading parts of the designs that are here and contributing to them.
To contribute you only need a text editor like notepad, and knowledge of writing markdown.

When reading these documents, always start with "index" any other documents are considered "suppporting"

## Values

Project values are important, acting as guideposts for tough decisions.
The are a couple of main values:

- Be Consistent
- Be Intuitive
- Be Clear

## Project Milestones

Pre-Alpha (Current)

- Pre-Alpha is a prototype stage, intended to be an extremely short version of the game's core concepts.

## Project Documentation Folder Structure

There are 4 main folders, each representing a fundamental part of the game or screen's the player can see.
Screens are indicated by appending "View" to the end.

The following folders are screens: 
- Core Aquarium View
- Shop VIew

The following folders are not screens, but are imperative to the game:
- Items
- Decorations

Within each main folder root, you will find a base description and valuable information in "index". 
**Other .md in root are considered important and also should be read.**
